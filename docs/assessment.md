## 三、答辩常见问题准备 / Anticipated QA Preparation

> 以下为学生自查用，不作为正式评估内容。

| 问题 | 要点提示 |
|------|---------|
| 为何选择零参数门控而非可学习门控？ | 避免引入额外推理开销；规则门控可解释性更强；消融实验验证效果相当 |
| StreamSpatial-VLM 与 VidCom² 的核心区别？ | VidCom² 基于纯2D外观相似度；本方法引入3D几何先验（深度方差+位姿不确定性），在空间推理任务上更精准 |
| 3倍加速的来源分解？ | Gate 减少3D触发次数（~60%帧跳过）+ Zip 减少token数量（50%）+ 增量缓存消除重复计算 |
| 方法在其他场景（非室内3D）的泛化性？ | 深度方差与位姿不确定性是通用几何信号；可扩展至自动驾驶、机器人导航等场景 |
| 若VGGT-1B不可用，框架是否仍有效？ | 可替换为 Depth-Anything-V2 提供深度先验；Gate 退化为纯2D语义门控，仍有加速效果 |
